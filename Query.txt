В базе данных MS SQL Server есть продукты и категории. Одному продукту может соответствовать много категорий, в одной категории может быть много продуктов. Напишите SQL запрос для выбора всех пар «Имя продукта – Имя категории». Если у продукта нет категорий, то его имя все равно должно выводиться.

Решение:
Допустим, продукты и категории - это таблицы Products и Categories. Таблица Products содержащие поля Name и Id. Для соединения продуктов и категорий используется соединительная таблица ProductCategory, содержащая поля ProductId и CategoryId. Тогда запрос будет выглядеть следующим образом:

Select P.Name, C.Name from Products P 
	left join ProductCategory PC on P.Id = PC.ProductId
	left join Categories C on PC.CategoryId = C.Id;
